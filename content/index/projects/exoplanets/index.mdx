An exoplanet data explorer for space enthusiasts. View random planets in great detail, search through the database of planets, or save interesting planets for later. All of these features with cool graphs to visualize size and mass of these planets. 

Built with React, Chakra-UI, Chart-JS, NASA's exoplanet archive and Node.
